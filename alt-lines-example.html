Alt Lines Badge - Working Example

Example 1: Spread Market WITH Alt Lines
17 Jan 07:00 PM - San Francisco 49ers at Seattle Seahawks
Spread
-110
// Badge appears because altSpreads array has items const spreadMarket = { spread: -3.5, altSpreads: [ { value: -3, odds: -110 }, { value: -4, odds: -105 } ] };
Example 2: Spread Market WITHOUT Alt Lines
17 Jan 07:00 PM - San Francisco 49ers at Seattle Seahawks
Spread
-110
// No badge appears because altSpreads is empty const spreadMarket = { spread: -3.5, altSpreads: [] };
